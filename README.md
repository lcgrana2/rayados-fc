# Rayados FC ⚽

Plataforma Fullstack para la gestión de equipos, estadísticas y control de acceso de la plantilla deportiva de Rayados FC.

## 🚀 Despliegue en Vivo
El proyecto se encuentra desplegado y listo para producción en: **[rayados-fc.vercel.app](https://rayados-fc.vercel.app)**

## 🛠️ Stack Tecnológico & Arquitectura
* **Framework:** Next.js (App Router) con TypeScript para un tipado robusto y renderizado optimizado.
* **Estilos y Componentes:** Tailwind CSS acoplado con Shadcn UI para una interfaz limpia, accesible y modular.
* **Backend as a Service (BaaS):** Supabase manejando el módulo de base de datos relacional y lógica de autenticación segura (`feature/auth`).

## 🤖 Desarrollo Asistido con IA (Eficiencia)
Este proyecto fue optimizado y acelerado aplicando flujos de trabajo avanzados de Ingeniería de Prompts y herramientas de Inteligencia Artificial (AI para Developers), logrando una velocidad de iteración superior en el modelado de datos de Supabase y el diseño de componentes UI modulares.

## ⚙️ Instalación Local
1. Clonar el repositorio.
2. Instalar dependencias con `npm install`.
3. Configurar las variables de entorno de Supabase en un archivo `.env.local`.
4. Ejecutar el servidor de desarrollo con `npm run dev`.
